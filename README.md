# Лабораторна робота №2 — Рядки в мові програмування Java
Виконала: Дубенок Діана  
група: ІМ-34  
номер в списку групи: 4  

### Тема
Рядки в мові програмування Java.

### Мета
Ознайомитися з рядками та використанням основних методів їх обробки в Java.  
Здобути навички оперування рядками, виконання операцій пошуку, сортування, заміни та аналізу тексту.

## Визначення варіанту

| Параметр | Обчислення | Значення | Пояснення |
|-----------|-------------|-----------|------------|
| **C3** | 4 % 3 | 1 | Тип даних — `String` |
| **C17** | 4 % 17 | 4 | Дія — у кожному реченні поміняти місцями перше та останнє слово |

## Опис реалізації

- Для роботи з рядками використовується **тип `String`**
- Програма розбиває текст на речення, потім кожне речення — на слова.
- Якщо в реченні більше одного слова — перше та останнє міняються місцями.
- Результат об’єднується назад у текст з оригінальними розділовими знаками.
- Реалізовано обробку можливих винятків (`try-catch`).
